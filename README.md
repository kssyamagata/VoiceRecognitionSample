# VoiceRecognitionSample
Prism for Xamarin.Forms（Prism.Forms）で作成した、音声認識のサンプルアプリです。

## 対応プラットフォーム
- iOS（バージョン10.0以上）
- Android

## 機能
音声を認識し、画面上に認識結果の文章を表示します。
